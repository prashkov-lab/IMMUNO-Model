# IMMUNO-Model
R and Octave code

data_fit_aggregated.R runs in R. It reads data points from appendix.xlsx.
It produces the output in files 2026_myDF_newXagg.dat, X=1,2,3...

fit_maximum.m and compute_distrib.m run in Octave. They both read from the files 2026_myDF_newXagg.dat,  X=1,2,3...

The appendix.xlsx contains also the model parameter estimates.
